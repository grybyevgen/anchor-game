# Инструкция по миграции: Переименование портов

## Важно!
После переименования портов в коде необходимо обновить названия портов в базе данных.

## Выполнение миграции

Выполните SQL-скрипт `migrate_rename_ports.sql` в вашей базе данных:

```sql
-- Миграция: переименование портов
UPDATE ports 
SET name = 'Порт "Нефтяной завод"'
WHERE name = 'Порт Владивосток';

UPDATE ports 
SET name = 'Порт "Провизионный завод"'
WHERE name = 'Порт Новороссийск';

UPDATE ports 
SET name = 'Порт "Завод Материалов"'
WHERE name = 'Порт Санкт-Петербург';
```

## Проверка

После выполнения миграции проверьте, что все порты переименованы:

```sql
SELECT id, name FROM ports;
```

Должны быть видны новые названия:
- Порт "Нефтяной завод"
- Порт "Провизионный завод"
- Порт "Завод Материалов"

## После миграции

1. Перезапустите backend сервер
2. Обновите страницу в браузере (Ctrl+F5 для полной перезагрузки)
3. Проверьте, что суда могут отправляться в порты
